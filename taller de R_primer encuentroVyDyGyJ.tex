\subsubsection{\texttt{While}}

El comando \texttt{while} sirve para construir bucles cuya continuidad depende de que se cumple una condición. La estructura es la siguiente:

\begin{center}
\shadowbox{
	\parbox[c][3cm][c]{8cm}{
		\texttt{while(}\textit{condición}\texttt{)}\\
		\{\\
		\textit{Bloque de código que queremos \newline
			que se ejecute en cada iteración.}\\
		\}
	}
}
\end{center}



Ejemplo: Calcular el factorial de 5

Idea: En cada iteración se van a realizar las siguientes operaciones\\
1*1=1\\
1*2=2\\
2*3=6\\
6*4=24\\
24*5=120\\

Observar que será necesario utilizar dos variables. Una de éstas realizará un recorrido entre 2 y 5 mientras que la otra irá guardando los resultados parciales que se obtienen al final de cada iteración. Finalmente, al completar el ciclo, en esta segunda variable se tiene el factorial del número que se está buscando.\\

\texttt{factorial=1}\\
\texttt{for(i in 1:5)\{}

   \texttt{factorial = factorial * i}

\}

Notar que la cantidad de iteraciones son fijas porque se está calculando el factorial de un número en particular. 
Esta situación cambia con la siguiente variante. Suponga que se quiere calcular el factorial de un número, el mayor posible, que no sea superior a 1200. En principio no se sabe cuál es el número, por lo tanto, la cantidad de iteraciones no es fija. En cada una de éstas vamos a calcular el factorial de 1,2,3, \ldots mientras se cumpla la condición requerida.

\bigskip

\begin{pro}
	Ejecute el siguiente código y observe que devuelve
\end{pro}

\texttt{n=1}\\
\texttt{factorial=1}\\
\texttt{while(factorial<1200)\{}

\texttt{factorial = factorial * n}\\
\texttt{n= n + 1}

\}\\

Una vez que se interrumpe el ciclo, ¿qué valores tienen las variables $n$ y factorial?

Veamos otro ejemplo algo más interesante. 
Quiero mostrar en pantalla los primeros 20 términos de la sucesión de Fibonacci:
\[
 0, 1, 1, 2, 3 , 5, 8, 13, 21, ...
\]
existe una fórmula que los genera iterativamente:
\[
 a_1 = 0; \ \ a_2=1; \ \ a_{n+2}=a_n + a_{n+1} 
\]
Esta última cuenta es la que debe aparecer en el bucle, veamos cómo hacerlo.
Puedo crear de antemano un vector
vacío de elementos para luego rellenar sus componentes,  la sentencia para este paso previo 
es: 
\[
 V=c()  .
\]
Ahora debo hacer asignaciones. Las dos primeras están fijas
V[1]=0
V[2]=1.
Las que siguen las debo generar con la fórmula, aquí comienza el proceso iterativo
entonces. Quiero generar V[3], V[4], ..., V[20] con $V[n+2] = V[n] + V[n+1]$. 
Entonces n variará entre 1 y 18. 
Dentro del bucle R hará las siguientes asignaciones: \newline
Para n=1
\[
 V[1+2] = V[1] + V[1+1]
\]
luego seguirá así:
\begin{align*}
  V[2+2] &= V[2] + V[2+1]  \ \ ( \text{ya lo tiene del paso anterior}) \\  
 V[3+2] &= V[3] + V[3+1] \\
V[4+2] &= V[4] + V[4+1] \\
& \vdots \\
V[18+2] &= V[18] + V[18+1]
\end{align*}
